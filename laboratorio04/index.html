Pasos para sumar 2 numeros

  1. Se definen las 2 variables a sumar
  2. Se inicializan en el codigo, o se le pide al usuario ingresarlas
  3. Se define la variable en la que se almacenara la suma
  4. Lo mostramos en pantalla

Pasos para buscar un elemento en un arreglo

  1. Almacenamos en elemento a buscar en una variable significativa
  2. En un bucle que se repita el tamaño del arreglo, recorremos todos los elementos
  3. Si el elemento a buscar esta en el arreglo, salimos del bucle y mostramos la posicion
  4. Si el elemento no esta en el arreglo, mostramos un mensaje de "No encontrado"
